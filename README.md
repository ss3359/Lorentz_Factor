# Lorentz Factor Simulation
## Overview
This project implements a numerical simulation of the Lorentz Factor to explore relativistic effects and chaotic behavior in a dynamic system. It demonstrates C++ and Python programming, numerical methods, and data visualization, making it a strong example of software engineering applied to complex scientific problems.

## Features

Numerical simulation of Lorentz Factor updates in C++

Python visualization of coordinate evolution and chaotic behavior

Generates data for analysis of system stability and divergence

Clean separation of simulation and visualization code for maintainability


## Skills Demonstrated 

Skills Demonstrated

Programming Languages: C++, Python

Numerical Methods: iterative calculations, data updates

Data Handling & Visualization: Pandas, Matplotlib

Problem-Solving: modeling complex physical systems, debugging large-scale simulations

Project Structure: modular code, readable scripts, documented workflow

## How To Run 

Compile and run the C++ simulation:

g++ Lorentz_Factor.cpp -o Lorentz_Factor
./Lorentz_Factor


Output will generate CppOutput.txt with updated coordinates.

Use Python notebook LFGraph.ipynb to visualize the data:

# In Jupyter Notebook
%run LFGraph.ipynb


Graphs will display coordinate evolution and patterns of chaos.
